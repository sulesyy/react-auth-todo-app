##  What I Learned in This Project

While developing this project, I didn’t just build a ToDo application — I deepened my understanding of React architecture and frontend development principles.

###  React State Management
- Managing dynamic data using `useState`
- Applying state immutability principles
- Updating nested state structures properly

###  useEffect Hook Usage
- Understanding the concept of side effects
- Working with dependency arrays
- Synchronizing state with localStorage

###  Data Persistence with LocalStorage
- Using JSON.stringify / JSON.parse
- Implementing user-specific data storage
- Managing persistent client-side data

###  Authentication Logic
- Building a basic user authentication system
- Storing the active user session
- Handling logout and session cleanup

###  Drag & Drop Architecture
- Implementing drag & drop with `@hello-pangea/dnd`
- Understanding source and destination logic
- Managing state updates across columns

###  Component & Routing Structure
- Separating components (Login & App)
- Navigation using React Router
- Organizing a clean and maintainable project structure

###  UI / UX Development
- Designing a Kanban board layout
- Creating a modal-based editing system
- Applying modern and responsive design principles

- Working with controlled components
- Handling events effectively in React
- Applying functional programming concepts
- Dynamic rendering using `.map()`
